SqlTools v4.1.4 to v4.2 Change Log

ISqlClass:

  • Added
    1. RowCount{get;}
    2. ColumnCount{get;}
  • Updated
    1. setConn now returns type bool instead of void
  • Deprecated
    1. getColumns() //use ColumnCount property
    2. getRows() //use RowCount property

MySqlClass:

  • Added
    1. RowCount{get;}
    2. ColumnCount{get;}
  • Updated
    1. setConn(...) now returns bool. It returns false on error, storing error into Errors rather than throwing an exception
    2. query(...) now returns false on error, storing error in Errors rather than throwing an exception
    3. command(...) now returns false on error storing error in Errors rather than throwing an exception
  • Deprecated
    1. getColumns() //use ColumnCount property
    2. getRows() //use RowCount property

MsSqlClass:

  • Added
    1. RowCount{get;}
    2. ColumnCount{get;}
    3. command(string)
    4. getSchema()
    5. getDataTable() //to be implemented in v4.2.1
    6. getDataSet() //to be implemented in v4.2.1
    7. Rows() as an IEnumerable<SqlRow>
      Exa.
      foreach(SqlRow row in sqlObj.Rows())
      { foreach(int x=0; x<row.Count; x++)
      {
      Console.Write("Col[{0}] = ", x, row[x].getString()); }       }
    8. free() //call this when you are done using the query, but still to keep the connection open. this will prevent some possible logic errors in the long run
    9. static List<string> Errors {get;}
    10. static List<string> Queries {get;}
    11. static List<string> QueriesFailed {get;}
  • Updated
    1. setConn(...) now returns bool. It returns false on error, storing error into Errors rather than throwing an exception
    2. query(...) now returns false on error, storing error in Errors rather than throwing an exception. Also, it now implements RowCount property.
    3. command(...) now returns false on error storing error in Errors rather than throwing an exception. Also, it now implements RowsEffected property
  • Deprecated
    1. getColumns() //use ColumnCount property
    2. getRows() //use RowCount property
